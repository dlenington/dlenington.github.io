My name is Dan and I am a creative web developer currently finishing college at the University of Wisconsin Madison.

I have always enjoyed creating art and writing music on the guitar, piano, and drums. After gaining an introduction to Java in a Computer Science class in college recommended by my brother, I found an interest in how computers work. I quickly gained more skills with coding and worked to create and iOS app with Objective C before becoming most interested in web development. I have created many websites and continue to develop in my free-time. I want to become a web developer to contribute my creative thoughts and coding ability to create stunning and useful websites. On this page you can find links to my web projects.
